# Guildframe keyword research

Research date: 2026-07-17

Source: Bing Webmaster Tools Keyword Research, global organic data, three-month
view, plus current result-page intent review.

## Findings

Broad theme demand is visible around the Shopify category:

| Related phrase | Three-month Bing impressions shown |
| --- | ---: |
| shopify themes | 3K |
| shopify theme store | 1.2K |
| shopify templates | 1.1K |
| free themes | 694 |

The exact niche phrase `board game shopify theme` did not have enough Bing data
to show a trend. It remains commercially relevant because it precisely describes
the product and buyer. Guildframe should pair it with the broader Shopify theme
language rather than replacing it with a generic theme page.

`shopify migration` and `shopify product page` also lacked enough Bing data for
a trend. Their result pages still show clear task intent:

- Shopify migration results emphasize migration guides, steps and checklists.
- Shopify product page results emphasize examples, templates, customization and
  optimization.
- Official Shopify documentation ranks prominently for both topics, making
  primary-source citation important.

## Guardrails

- Use the exact niche phrase where it naturally identifies the product or task.
- Do not create separate pages for synonyms with the same intent.
- Keep trust pages focused on brand, authorship and editorial standards instead
  of forcing commercial keywords into them.
- Review real Search Console and Bing queries before adding another landing page.

## Supporting intent evidence

Current official result coverage confirms distinct task intent for the new pages:

- Shopify documents preorders as a purchase option and separately documents
  product, variant, metafield and bundle behavior.
- Kickstarter documents Late Pledges, pledge-manager sequencing and backer-report
  roles separately from permanent ecommerce.
- Kickstarter's 2024 Games retrospective provides an authoritative data source
  for tabletop funding statistics rather than relying on unsupported market claims.

These pages were added because they answer different operational questions. They
are not synonym pages for the existing migration or theme landing pages.
